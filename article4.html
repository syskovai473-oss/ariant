Термины

1.Компилятор
— это программа, которая переводит человекочитаемый код на языке программирования в машинный код, понятный компьютеру.
2.Язык программирования
— это формальная знаковая система, состоящая из набора правил (синтаксиса и семантики), позволяющая человеку давать инструкции компьютеру для выполнения задач.
3.Программа
— это набор инструкций, написанных на языке, который понимает компьютер, чтобы выполнить определённую задачу.
4.Подпрограмма
— это набор инструкций, написанных на языке, который понимает компьютер, чтобы выполнить определённую задачу. это именованный или идентифицированный фрагмент кода, который выполняет определённый набор действий и может быть вызван из других частей программы многократно.
5.Код
— это набор инструкций, написанных на языке, который понимает компьютер, чтобы выполнить определённую задачу.определенный набор символов или определенную последовательность символов, которые несут в себе некую зашифрованную информацию, передают смысл по конкретному алгоритму.
